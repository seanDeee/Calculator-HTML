# Calculator
A Simple Calculator with some Trigonometric Functions


## Built With

* [HTML] 
* [CSS]
* [JAVASCRIPT] & [JQUERY]

## Usage

* Calculate Basic Math: Add, Subtract, Multiply and Divide. <br />
* Calculate some Trigonometric Functions: Cos, Sin, Tan and Log.
